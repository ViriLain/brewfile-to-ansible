# Brewfile to Ansible Converter

Have you ever manually installed a bunch of stuff on your computer using Brew, but then realized you should probably convert your installs to Ansible? Then this is for you!

A Brewfile to Ansible playbook converter. A small piece of code to fix a very specific problem of mine. 

[Brewfile Usage](https://homebrew-file.readthedocs.io/en/latest/usage.html)
